## Contributing

Please add unit tests for new functionality.

Make sure tests are green:
```
composer test
```

You can check coverage with
```
composer test-coverage
```

Running CS sniffer:
```
composer cs-check
```

Running CS fixer:
```
composer cs-fix
```
